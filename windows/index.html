C盘下系统文件夹的含义
2024-01-01 ~ 2024-01-01
  • C:\Program Files：这是 64 位软件的默认安装目录，所有用户都能使用，程序安装到这里需要管理员权限。
  • C:\Program Files (x86)：用于存放 32 位的软件，同样需要管理员权限才能安装。
  • C:\Users\ 用户名 \AppData\Roaming：通过环境变量 % AppData% 可以访问，用来存放当前登录用户所产生的数据，对其他用户不可见。部分软件可能会将一些用户配置文件、缓存等存储在此处。
  • C:\Users\ 用户名 \AppData\Local：可通过环境变量 % LocalAppData% 访问，该目录主要存放仅限本地使用的数据，一般个人使用的软件可安装在 C:\Users\ 用户名 \AppData\Local\Programs 下面。
  • C:\ProgramData：这里存放的数据是公用的，若希望软件的数据在所有用户之间保持一致，程序生成的数据可存放在此。
  • C:\WindowsApps：这是通过 Windows 应用商店下载安装的应用程序所在目录，是一个隐藏文件夹，且需要相应权限才能访问。
