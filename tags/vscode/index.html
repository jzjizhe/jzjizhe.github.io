行号设置

VS Code 内置行号有三种模式：

  • "off"：不显示行号
  • "on"：绝对行号
  • "relative"：相对行号
    设置方式（以"relative"为例）：
  1. 打开设置（Ctrl + ,）
  2. 搜索 line numbers
  3. 找到 Editor: Line Numbers，选择 relative
