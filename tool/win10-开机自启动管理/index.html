📁 方法一：将程序添加到「启动」文件夹（推荐）

这是最常用、最简单的开机自启动方式。

✅ 添加步骤：

  1. 按下 Win + R 打开 运行 窗口。

  2. 输入以下命令并回车：

    1

    shell:startup

  3. 系统会打开 启动文件夹，将你想要开机自启的程序 快捷方式（.lnk） 放入此文件夹即可。

    • 如果没有快捷方式：右键空白处 → “新建” → “快捷方式” → 浏览并选择目标程序的 .exe 文件。

💡 说明：

  • 放在这里的程序会在 当前用户登录时 自动启动。

  • 如果想让 所有用户 开机自启，可以在运行中输入：

    1

    shell:common startup

🔧 方法二：通过「任务管理器」禁用自启程序

有些软件会自动注册开机启动，即使你没有放入启动文件夹，它们仍然会开机运行。可以用任务管理器快速关闭。

✅ 禁用步骤：

  1. 按下快捷键 Ctrl + Shift + Esc 打开 任务管理器。

  2. 切换到顶部的 “启动” 选项卡。

  3. 在列表中找到你不想自动启动的程序。

  4. 右键该程序 → 选择 “禁用”。

💡 提示：

  • 状态列显示 “已启用” 的程序会在开机时自动运行。

  • “启动影响” 一栏可以帮助你判断该程序对开机速度的影响。

  • 禁用后不会卸载程序，只是让它不再自动启动。

🧰 进阶：查看和管理“隐藏的自启程序”

即使在 shell:startup 和任务管理器里没看到，软件依然能自启？
那是因为它们使用了 注册表 或 计划任务 等更底层的方式。

🪛 方法三：查看注册表自启项

大多数软件会把自启信息写入注册表：

  • 当前用户（仅对你生效）：

    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run

  • 所有用户（系统级）：

    HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run

✅ 查看步骤：

  1. 按下 Win + R，输入：

    1

    regedit

  2. 按路径定位到上述位置

  3. 在右侧窗口查看或删除你不需要的程序项（删除前建议先备份注册表）

⚠️ 注意：

  • 修改注册表有风险，请谨慎操作

  • 删除错误项可能导致系统或软件异常

📆 方法四：检查计划任务中的自启程序

某些软件会通过「任务计划程序」自启（可延迟启动、隐藏后台运行等）。

✅ 查看步骤：

  1. 按下 Win + S，搜索“任务计划程序”并打开

  2. 在左侧选择 任务计划程序库

  3. 浏览中间列表，查找与软件名称相关的任务

  4. 不需要的任务：右键 → “禁用” 或 “删除”

💡 常见例子：显卡驱动、同步软件、杀毒软件等。

🛠️ 高级技巧：使用 Autoruns 一站式管理所有自启

微软官方提供了一款非常强大的工具 👉 Autoruns for Windows

✅ 功能：

  • 列出 所有 自启项目（启动文件夹、注册表、计划任务、服务等）

  • 支持一键禁用/删除

  • 帮助排查“隐藏自启”问题

📌 总结

操作工具适用场景
✅ 添加自启shell:startup 文件夹手动让程序随开机运行
🔧 禁用自启任务管理器 → 启动关闭常规自启软件
🪛 查看注册表regedit查找注册表隐藏自启
📆 检查计划任务任务计划程序查找延迟/后台自启程序
🔍 全面扫描Autoruns 工具一次性查看所有自启来源

💡 建议：

  • 常用软件用「启动文件夹」自启即可

  • 系统性能慢时，先在「任务管理器」禁用不必要自启

  • 想完全掌控启动行为，用 Autoruns 是最彻底的办法
